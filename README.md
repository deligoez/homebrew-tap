# homebrew-tap

Homebrew tap for [deligoez](https://github.com/deligoez) CLI tools.

## Install

```bash
brew install deligoez/tap/tp
brew install deligoez/tap/rp   # alias for deligoez-rp; installs the `rp` binary
brew install deligoez/tap/hc   # alias for deligoez-hc; installs the `hc` binary
```

## Upgrade

Homebrew does not refresh third-party taps during `brew upgrade`, so update first:

```bash
brew update
brew upgrade tp deligoez-rp deligoez-hc
```

## Formulas

| Formula | Alias | Binary | Description |
|---------|-------|--------|-------------|
| [tp](https://github.com/deligoez/tp) | -- | `tp` | Spec-to-task lifecycle manager for AI coding agents |
| [deligoez-rp](https://github.com/deligoez/rp) | `rp` | `rp` | Repo manager CLI -- organize, sync, and bootstrap your Developer workspace |
| [deligoez-hc](https://github.com/deligoez/hc) | `hc` | `hc` | Hunk-based atomic git commits for AI agents |

The `rp` and `hc` formulas are named `deligoez-rp` / `deligoez-hc` to avoid name
collisions; the tap-qualified aliases above make the short names work anyway.
Formula files are generated by GoReleaser on each release -- do not edit by hand.
